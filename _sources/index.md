# CR-Suite

The `CR-Suite` includes the following libraries:

```{list-table}
:header-rows: 1
:widths: 20 60 10 10

* - Name
  - Features
  - Repository
  - Docs
* - CR-Nimble
  - Common linear algebra and signal processing routines
  - [GitHub](https://github.com/carnotresearch/cr-nimble)
  - [Docs](https://cr-nimble.readthedocs.io/en/latest/)
* - CR-Wavelets
  - Discrete and continuous wavelets
  - [GitHub](https://github.com/carnotresearch/cr-wavelets)
  - [Docs](https://cr-wavelets.readthedocs.io/en/latest/)
* - CR-Sparse
  - Sparse Signal Processing, Compressive Sensing, Sparse Recovery, Convex Optimization
  - [GitHub](https://github.com/carnotresearch/cr-sparse)
  - [Docs](https://cr-sparse.readthedocs.io/en/latest/)
```
